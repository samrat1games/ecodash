## Тип моего проекта:
> EcoDash — модульный экологический веб-портал с трекерами привычек, интерактивными челленджами, калькулятором углеродного следа и VPN-менеджером. Фронтенд на чистом JS + Tailwind, десктоп VPN-клиент на Python с GUI.
## Библиотеки, которые я буду использовать
- customtkinter — современный GUI для VPN-клиента (тёмная тема, стилизованные виджеты)
- Xray-core — ядро для VPN-подключений (VLESS/ Reality протоколы)

Современный GUI на Python — CustomTkinter Docs .
