
🍃 ECODASH

ECODASH — это комплексный веб-проект, объединяющий экологические инициативы, трекеры привычек и информационные лендинги. Сайт разбит на независимые модули для удобной навигации и расширения функционала.
🛠 Состав проекта
🧬 Эко-система

    ecochelendg — интерактивные челленджи для пользователей.

    recycling — информационный раздел по правильной переработке отходов.

    calc.html — калькулятор экологического следа или расхода ресурсов.

📊 Инструментарий

    trecker — графический интерфейс для отслеживания ежедневного прогресса.

    timer — встроенный таймер для фокусировки на задачах.

    shop — страница витрины товаров.

📂 Технические модули

    reklama — хранилище графических ассетов и баннеров (67.png, brainrot.png, fpi.png).

    vpnsite — лендинг для раздачи клиентского ПО и документации.

    vpn — внутренняя директория с конфигурациями и скриптами управления.
